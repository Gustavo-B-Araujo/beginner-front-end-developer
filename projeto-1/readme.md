# Projeto 1: Checkpoint 1 - Projeito Carreira

## Descrição
Problema: Sua dupla foi contratada para criar a estrutura de um portal de carreira profissional. O projeto consiste em uma página principal (index.html) que funciona como um sumário, e três páginas internas detalhadas sobre: Formação, Habilidades e Contatos. O objetivo é demonstrar organização de arquivos, uso correto de caminhos relativos e controle de versão via Git.
