# Projeto 2: Introdução ao JavaScript

## Descrição

Este projeto consiste em uma página HTML simples que utiliza uma tag `<script>` para demonstrar conceitos básicos de JavaScript.
O objetivo é praticar e consolidar os fundamentos da linguagem.

## Conceitos abordados

O script apresenta exemplos práticos dos seguintes tópicos:

* Comentários em JavaScript
* Variáveis (`let`)
* Tipos de dados
* Operadores aritméticos
* Estruturas condicionais (`if`, `else`)
* Estruturas de repetição (`for`, `while`)
* Funções
* Coleções (arrays e objetos)


## Como executar

1. Acesse a pasta do projeto:

   ```bash
   cd projeto-2
   ```

2. Abra o arquivo `index.html` em seu navegador.

## 📚 Objetivo de aprendizado

Este projeto tem como foco a introdução à lógica de programação com JavaScript, servindo como base para projetos mais avançados.
